## 🔍 진행 방식

- 최종 코딩 테스트 대비 스터디 개인 저장소입니다.
- 5시간 동안 각 주차의 미션을 최대한 구현합니다.
- 미션 진행 이후 코드리뷰를 위한 PR을 남깁니다.
- 주어진 시간내에 돌아가는 프로그램을 만드는 것을 목표로 합니다.
- 테스트 통과를 마친 후, 클린 코드, 리팩터링, 테스트 등을 보완합니다.
